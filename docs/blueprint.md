# **App Name**: The Law Leads PWA

## Core Features:

- Hero Section with Dynamic Headline: A visually appealing hero section with a dynamic headline, emphasizing key selling points like 'Best Judiciary & Law Entrance Coaching'.
- Interactive Course Display: Display the courses in an interactive format with clear distinctions between Judiciary, CLAT PG, LLM Entrance, and Law Notes.
- Faculty Showcase: A dedicated section to showcase faculty profiles with images and brief descriptions to build trust and credibility.
- WhatsApp Inquiry Integration: Implement a 'Book Free Demo' form that captures user details and redirects the data to WhatsApp for direct inquiries.
- SEO-Optimized Blog: Integrate a blog section with SEO-optimized articles and content related to law, judiciary exams, and coaching.
- FAQ with Schema Markup: An FAQ section that answers common questions, incorporating schema markup to improve SEO ranking.
- AI-Powered Content Suggestion Tool: AI-powered tool to suggest related content that users might be interested in based on what section of the site they are visiting. This improves user engagement and can highlight services they may have overlooked.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) to convey trust and professionalism.
- Background color: Light desaturated blue (#E8EAF6) to complement the dark blue without being distracting.
- Accent color: Gold (#FFD700) for highlights, calls to action, and a premium feel.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, objective look.
- Glassmorphism cards to give depth and a modern aesthetic.
- Sticky navbar for easy navigation and accessibility.
- Smooth scroll animations to provide a fluid user experience.
- Animated Hero Background for eye-catching visual engagement.
